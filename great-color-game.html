The Great Color Game

para RGB experts

Stack de tecnologias utilizadas:

HTML, CSS, JavaScript

GitHub: https://github.com/dmmello1989/thegreatcolorgame

Um simples jogo de descobrir a que cor o código RGB gerado aleatoriamente pertence. É possível escolher duas dificuldades: Easy, a mais fácil, com 3 respostas possíveis; Hard, a mais difícil, com 6 respostas possíveis.
