Stack de tecnologias utilizadas:

Be The Hero é um projeto desenvolvido durante a Semana OmniStack 11.0 da RocketSeat, ministrado pelo Diego Fernandes.

Página de Login

Be The Hero é um aplicativo que busca auxiliar financeiramente, através de doações, entidades e organizações não governamentais.

O primeiro passo é cadastrar uma ONG e utilizar o código de login para entrar no aplicativo e visualizar o dashboard.

Registrar ONG Dashboard

Com a ONG logada, podemos registrar novos casos, especificando o assunto e os detalhes, bem como o valor de contribuição.

Registrar Novo Caso

Desenvolvemos a aplicação do zero utilizando a stack poderosa de Node.js e Express para o backend, React no frontend e aplicativo mobile com React Native.

Para o banco de dados, escolhemos usar SQLite 3 em conjunto com o Knex.js (SQL Query Builder).
